# Book App 

This project uses Shadcn as a component library
https://ui.shadcn.com/docs/components

Tailwind Css for styling
https://tailwindcss.com/docs/utility-first


## Installation

Install book-app with npm

```bash
  npm install
```

run the project

```bash
  npm start
```
